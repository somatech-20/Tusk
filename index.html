Project Purpose

This project, the "Task Management Application," is a web-based platform designed to facilitate efficient task and project management. Its purpose is to help users manage and track their tasks effectively by providing a structured system for project planning, task assignment, and progress monitoring. By combining task management with role-specific functionalities, the application aims to improve productivity, ensure efficient project execution, and promote better team collaboration.

Core Concept: The 3 Roles

The application's architecture is built on three distinct user roles, each with specific permissions and functionalities designed to streamline workflow and create a clear hierarchy of operations.

Technology Stack

The tools and technologies used to build our application or rather our version

React Native

An open-source JavaScript-based framework used to create natively-rendered mobile applications for iOS and Android platforms.

TypeScript

A superset of JavaScript that adds static typing, widely supported in React Native for improved code quality and developer experience.

Firebase

A cloud-based backend platform by Google, commonly integrated with React Native for authentication, real-time database, cloud storage, and serverless functions.

Firestore

A scalable, serverless NoSQL database from Google, integrated with Firebase to enable real-time data synchronization, offline support, and complex querying for mobile and web applications.

Expo

A free and open-source platform for React Native that simplifies development with pre-built components, tools like Expo Go for instant testing, and cloud services for building and deploying apps.

TailwindCSS

A utility-first CSS framework that can be integrated with React Native using libraries like NativeWind to enable rapid, class-based styling across mobile platforms.

pnpm

A fast, disk-efficient package manager that uses symlinks and a central store, compatible with React Native when configured properly to handle Metro's symlink limitations.

Conclusion & Future

A summary of the project's achievements and potential future enhancements.

Project Conclusion

The Task Management Application successfully provides a powerful tool to manage and track tasks across multiple hierarchies. Its robust backend and clear, role-based frontend ensure smooth project management, allowing Administrators, Project Managers, and Employees to handle their responsibilities effectively. The application enhances productivity and communication by providing a clear structure, and its user-friendly interface ensures all users can navigate the system with ease.
